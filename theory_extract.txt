ﾂｧ Given a relation R defined on some set S, R is an equivalence
relation if (and only if), for all, x, y, and z elements of S:
ﾂｧ R is reflexive, i.e., xRx
ﾂｧ R is symmetric, i.e., if xRy, then yRx
ﾂｧ R is transitive, i.e., if xRy and yRz, then xRz
ﾂｧ An equivalence relation, R, induces a partition on the set S, where a
partition is a set of subsets of S such that:
ﾂｧ The intersection of any two subsets is empty, and
ﾂｧ The union of all the subsets is the original set S
ﾂｧ Note that the intersection property assures no redundancy, and the
union property assures no gaps.
